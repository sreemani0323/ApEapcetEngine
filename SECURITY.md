# Security Policy

## Supported Versions

The EAPCET Intelligence Engine currently supports the following versions for security updates. 

| Version | Supported          |
| ------- | ------------------ |
| 1.x     | :white_check_mark: |

## Reporting a Vulnerability

We take the security of the EAPCET Intelligence Engine seriously. If you believe you have found a security vulnerability, please report it to us as described below.

**Please do not report security vulnerabilities through public GitHub issues.**

Instead, please email the maintainer directly or use GitHub's private vulnerability reporting feature if enabled on the repository.

### What to include in your report

Please include the following details in your report:
- A description of the vulnerability and its impact.
- Steps to reproduce the issue.
- Any relevant logs, screenshots, or code snippets.
- (Optional) Suggested mitigation or remediation steps.

### Response Timeline

We will try to acknowledge receipt of your vulnerability report within 48 hours and strive to provide a regular update on our progress until the issue is resolved.

Thank you for helping keep this project secure!
